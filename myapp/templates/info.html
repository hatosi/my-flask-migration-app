Dự án Phân tích Tỷ lệ Di cư Việt Nam - DAP391m Data Science Project

👋

LỜI MỞ ĐẦU

Xin chào các bạn! Chúng tôi là nhóm sinh viên chuyên ngành AI đến từ Trường Đại học FPT Đà Nẵng. Trang web này chính là kết quả của bài tập nhóm môn DAP391m - Data Science Project. Đây là trang web dự đoán kết quả của một trong những vấn đề thuộc lĩnh vực dân số - tỷ lệ di cư thuần tại các tỉnh thành của Việt Nam (theo phân cấp 63 tỉnh thành cũ).

📊

TỔNG QUAN DỰ ÁN

Trong dự án này, chúng tôi thực hành các kiến thức được học của bộ môn Data Science đồng thời áp dụng mô hình Học máy (Machine Learning) để giải quyết bài toán.

📋 Dữ liệu thu thập

🌍 Thông tin địa lý
  • Khu vực
  • Tên tỉnh thành
  • Diện tích
  • Trực thuộc Trung ương
👥 Thông tin dân số
  • Dân số theo năm
  • Mật độ dân số
  • Tỉ lệ di cư thuần (đầu ra)
🌤️ Thông tin khí hậu
  • Nhiệt độ trung bình năm
  • Tổng lượng mưa cả năm
  • Tổng lượng tuyết rơi
  • Tổng số giờ mưa
  • Tổng số giờ nắng
💰 Thông tin kinh tế
  • GRDP bình quân đầu người
  • Thu nhập bình quân/tháng
🚢 Hạ tầng giao thông
  • Số sân bay dân dụng
  • Loại cảng đường thủy
📅 Thời gian
  • Năm: từ 2004 đến 2024
  • 20 năm dữ liệu lịch sử

📚 Nguồn dữ liệu

Các dữ liệu được thu thập chủ yếu từ Cục Thống kê, Bộ Tài chính. Ngoài ra dữ liệu còn được thu thập từ các văn bản báo cáo kinh tế - xã hội của các tỉnh thành.

⚙️ Quy trình thực hiện

1
Thu thập dữ liệu từ các nguồn chính thống
2
Xem xét và xử lý các dữ liệu thiếu
3
Trực quan hóa dữ liệu để tìm ra quy luật
4
Lựa chọn mô hình Học máy thích hợp
5
Xây dựng và huấn luyện mô hình
6
Triển khai trên web
🎯

MỤC ĐÍCH DỰ ÁN

Dự án được ra đời với mục đích ứng dụng các kiến thức chuyên ngành đã được học để giải quyết bài toán thực tế trong xã hội. Bên cạnh đó, chúng tôi hy vọng trang web này có thể là một công cụ tham khảo hữu ích về số liệu di cư cho mọi người. Đặc biệt hơn cả, chúng tôi hy vọng rằng dự án này có thể mở rộng để ứng dụng trong thống kê các chỉ số kinh tế xã hội của Việt Nam.

⚡

CÔNG NGHỆ SỬ DỤNG

Dự án được xây dựng trên nền tảng công nghệ hiện đại, đảm bảo hiệu suất cao và trải nghiệm người dùng tối ưu:

🚀 Backend & Machine Learning

🐍 Python Ecosystem
  • Python 3.11+ - Core language
  • Flask - Web framework
  • Jinja2 - Template engine
  • Werkzeug - WSGI utilities
🤖 Machine Learning
  • Scikit-learn - ML algorithms
  • Pandas - Data manipulation
  • NumPy - Numerical computing
  • Matplotlib - Data visualization
📊 Data Processing
  • Jupyter Notebook - Development
  • Seaborn - Statistical plots
  • XGBoost - Gradient boosting
  • Random Forest - Ensemble methods

🎨 Frontend & UI/UX

🌐 Web Technologies
  • HTML5 - Semantic markup
  • CSS3 - Advanced styling
  • JavaScript ES6+ - Interactivity
  • Responsive Design - Mobile-first
✨ Modern CSS Features
  • CSS Grid & Flexbox - Layout
  • CSS Gradients - Visual effects
  • CSS Animations - Smooth transitions
  • Media Queries - Responsiveness
🎯 UX/UI Design
  • Material Design principles
  • Glassmorphism effects
  • Micro-interactions
  • Progressive Web App features

🛠️ Development & Deployment

⚙️ Development Tools:

  • Git & GitHub - Version control
  • VS Code - Code editor
  • Postman - API testing

🌐 Deployment:

  • Gunicorn - WSGI server
  • Nginx - Reverse proxy
  • Cloud hosting platforms

📈 Performance:

  • Lazy loading techniques
  • CSS/JS minification
  • Image optimization

🏗️ Architecture Highlights

Website được thiết kế theo MVC Pattern với RESTful API, sử dụng Template Engine để render dynamic content. Giao diện được tối ưu với Progressive Enhancement và Graceful Degradation để đảm bảo tương thích trên mọi thiết bị.
